# Automatic Parking System using STM32

## Overview

This project is an automatic parking system developed using the STM32 microcontroller. The system automatically controls vehicle entry and exit and monitors parking availability.

## Features

* Automatic vehicle detection
* Entry and exit gate control
* Parking slot monitoring
* STM32 embedded control
* Sensor-based automation

## Hardware Used

* STM32 Microcontroller
* IR Sensors / Ultrasonic Sensors
* Servo Motor
* LCD Display (if used)
* Power Supply Module

## Software & Tools

* STM32CubeIDE / Keil C
* Embedded C Programming

## System Operation

1. Vehicle arrives at the gate.
2. Sensors detect the vehicle.
3. STM32 checks parking availability.
4. Barrier gate opens automatically.
5. Parking status is updated.

## Project Purpose

This project was developed to improve knowledge in embedded systems and automation using STM32.
